Smart Indoor Navigation System

Google Maps-style indoor navigation across campus buildings using Wi-Fi zone localization.

Try Interactive Route Demo

Overview

Navigate classrooms, labs, cabins, and offices inside multi-floor buildings using existing Wi-Fi routers. This platform reduces confusion, improves accessibility, and offers emergency routing support.

Students

Find classrooms, labs, and departments quickly.

Faculty

Locate cabins and meeting rooms with minimal effort.

Visitors

Get turn-by-turn indoor guidance without prior familiarity.

Admin

Manage maps, routers, and route graph definitions.

Core Features

  • Wi-Fi router-based zone detection
  • Shortest path routing with Dijkstra algorithm
  • Multi-floor and multi-building support
  • Live location updates every 3–5 seconds
  • Blue path route visualization
  • Accessibility mode
  • Emergency exit routing
  • QR fallback for backup positioning
